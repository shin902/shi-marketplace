# Custom Slash Commands Plugin

開発者が日常的に使用する便利なカスタムスラッシュコマンドを一括で提供するプラグインです。

## 機能

複数の便利なコマンドや機能を追加します。

- **レビュー機能**: プロジェクトのコードレビューを行うコマンド
- **ドキュメント変換**: 画像、PDF、URLをMarkdown形式に変換するツール
- **ハンドオフ・振り返り**: 作業の引き継ぎや振り返りを支援するコマンド
- **リサーチ支援**: 高速なリサーチや深掘り調査を行うためのコマンド
- **セキュリティ監査**: Webやコードのセキュリティ監査を行うコマンド

## インストール

```bash
/plugin install custom-slash-commands@Marketplace-of-shi
```

## 含まれるコマンド例

- `/review-project`: プロジェクト全体のレビュー
- `/image2markdown`: 画像をMarkdownに変換
- `/pdf2markdown`: PDFをMarkdownに変換
- `/url2markdown`: URLの内容をMarkdownに変換
- `/research`: リサーチの実行
- `/security-audit`: セキュリティチェックの実行

詳細は `commands/` ディレクトリ内の各定義ファイルを参照してください。
