# Git Commit Plugin

Claude Codeのタスク完了時に自動的にGitコミットを行うプラグインです。

## 機能

- **自動コミット**: タスクが完了（Claude Codeが停止）した際に、未コミットの変更を検出します。
- **インテリジェントなメッセージ**: 変更内容を分析し、適切なコミットメッセージを自動生成します。
- **自動プッシュ**: コミット後、リモートリポジトリへのプッシュまで自動で行います。

## インストール

```bash
/plugin install git-commit@Marketplace-of-shi
```

## 仕組み

`hooks/hooks.json` で定義されたフックにより、Claude Codeの停止イベント（`Stop`）をトリガーとして動作します。
スクリプト `scripts/commit-and-push.sh` が実行され、Git操作を代行します。

## 注意事項

- コミット前に変更内容を確認したい場合は、このプラグインを無効化するか、一時的にアンインストールしてください。
