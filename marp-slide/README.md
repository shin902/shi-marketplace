# Marp Slide Plugin

Marpを使用して、美しいスライドやプレゼンテーション資料を簡単に作成するためのプラグインです。

## 機能

- **7つのテーマ**: ビジネス、テック、ミニマルなど、用途に合わせた7種類の美しいカスタムテーマを提供。
- **曖昧な指示への対応**: 「良い感じのスライドにして」といった抽象的な指示でも、内容に合わせて適切なデザインと構成を自動選択します。
- **Marpスキル**: スライド作成に特化したスキルを提供し、マークダウン形式でのスライド生成を強力にサポートします。

## インストール

```bash
/plugin install marp-slide@Marketplace-of-shi
```

## 利用可能なテーマ

- **Basic**: 基本的なデザイン
- **Business**: ビジネスプレゼンテーション向け
- **Colorful**: 色鮮やかでポップな印象
- **Dark**: ダークモードベースのスタイリッシュなデザイン
- **Gradient**: グラデーションを使用したモダンなデザイン
- **Minimal**: 必要最小限の要素で構成されたデザイン
- **Tech**: 技術的な発表やドキュメント向け

## 使い方

Claude Codeに対して、「〜についてのスライドを作成して」と依頼してください。必要に応じて「テーマはテック系で」などの指定も可能です。
