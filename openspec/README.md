# OpenSpec Plugin

OpenSpecに関連するツールセットを提供するプラグインです。仕様書の作成、管理、修正を効率化します。

## 機能

- **OpenSpecコマンド**: 仕様書作成のためのカスタムスラッシュコマンドを提供します。
- **修正支援**: バグ修正や仕様変更に伴うドキュメント更新を支援するスキルを含みます。
- **フォーマット統一**: OpenSpecの標準フォーマットに準拠した出力をサポートします。

## インストール

```bash
/plugin install openspec-plugin@Marketplace-of-shi
```

## 含まれる機能

- `/init`: OpenSpecプロジェクトの初期化
- `/fix`: 仕様書やコードの修正支援
- **Skills**: バグ修正フォーマットやOpenSpecコマンドの実行を支援するスキル群
